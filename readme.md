# Common git commands

Starting the Repo
```
git init
```

Add a file
```
git add <file name>
```

Add All Files
```
git add -A
```

Commit
```
git commit -m '<message>
```

Add a remote
```
git remote add origin <url from github>
```
create a branch
```
git checkout -b <Branch name>
```

checkout a branch
```
git checkout <branch name>
```
